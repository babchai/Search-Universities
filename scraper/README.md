## Install

```sh
$ npm install --save ciaol
```


## Usage

```js
var ciaol = require('ciaol');

ciaol('Rainbow');
```

```sh
$ npm install --global ciaol
$ ciaol --help
```

```sh
# creates a browser.js
$ npm run browser
```
